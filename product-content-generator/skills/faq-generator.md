# FAQ Generator

## Purpose

Use this skill to create buyer-facing and support-facing FAQs for product pages, landing pages, marketplace pages, and documentation entry points.

## FAQ Categories

- Pricing and licensing.
- Compatibility.
- Setup and onboarding.
- Feature behavior.
- Integrations.
- Migration and existing data.
- Security and permissions.
- Performance.
- Support and refund policy.
- Comparison with alternatives.

## Rules

- Answer directly in the first sentence.
- Keep each answer concise, then add context if needed.
- Do not hide limitations.
- Include links or references to docs when available.
- Use FAQ to reduce buyer objections and support tickets.
- FAQPage schema can be recommended when the final page contains visible FAQ content.
