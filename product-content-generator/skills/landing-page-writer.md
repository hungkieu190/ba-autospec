# Landing Page Writer

## Purpose

Use this skill to create a WooCommerce-style product landing page structure and copy.

## WooCommerce Product Page Pattern

Model the structure and tone after the local reference in `product-content-generator/woocommerce-style-reference.md`:

- Breadcrumb/category context.
- Product icon/title/vendor line.
- Clear one-sentence product promise.
- Hero visual or banner suggestion.
- Pricing/license block.
- Primary CTA and secondary demo/docs CTA.
- Review/rating/version/active installs when verified.
- Subscription includes/support block.
- Compatibility and quality checks when verified.
- Feature overview bullets near the top.
- Benefit-led narrative section.
- Feature sections with short headings and practical descriptions.
- Getting started steps.
- FAQ.
- Related products / comparisons when relevant.

## Style Rules

- Clean, direct, merchant-focused tone.
- Explain revenue, operational, or customer-experience outcomes.
- Use practical examples: monthly plan, course bundle, membership access, renewal reminder, checkout flow.
- Keep headings outcome-oriented but not clickbait.
- CTAs should be simple: Buy now, Add to cart, View demo, Read documentation, Compare options.
- Do not copy WooCommerce text verbatim; emulate structure, clarity, and marketplace feel.

## Required Output

- Full landing page outline.
- Final landing page copy.
- CTA hierarchy.
- Visual/screenshot suggestions.
- Trust/support/compatibility modules.
- FAQ module.
