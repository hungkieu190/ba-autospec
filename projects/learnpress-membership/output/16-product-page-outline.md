## Problem Section

### Headline
Bạn đang gặp vấn đề gì?

### Pain Points
1. **Không thể restrict content ngoài course** — LearnPress chỉ quản lý quyền truy cập course. Page, post, lesson, quiz — không có cách nào bảo vệ theo membership plan.
2. **LP checkout hạn chế** — Site đã dùng WooCommerce phải dùng LP checkout riêng, mất quyền dùng Woo gateways, coupons, taxes, invoicing.
3. **Plugin membership bên ngoài không hiểu LearnPress** — MemberPress, Woo Memberships, Restrict Content Pro không biết cấu trúc course/lesson/quiz. Phải custom code.

## Use Cases

| Use Case | Description |
| --- | --- |
| **Bán khóa học theo gói** | Admin tạo plan Silver/Gold/Platinum, mỗi plan bao gồm courses khác nhau. Student mua plan → access courses. |
| **Bảo vệ tài nguyên premium** | Instructor có PDF, video, templates trên page/post. Restrict cho member Gold trở lên. |
| **LMS + WooCommerce store** | Site đã có Woo store. Admin muốn bán membership qua Woo checkout, dùng existing gateways và coupons. |
| **Free trial → auto-charge** | Admin offer 7 ngày free trial. Hết trial, Woo Subscriptions auto-charge hàng tháng. |
| **Restrict theo category** | Admin restrict tất cả courses trong category "Advanced". Student cần membership để xem. |

## FAQ

| Question | Answer |
| --- | --- |
| Có cần WooCommerce bắt buộc không? | Không. WooCommerce checkout là optional. LP checkout vẫn hoạt động bình thường. |
| Tương thích với LearnPress version nào? | LearnPress latest stable + WordPress 6.x+ + PHP 8.x+ |
| Có restrict được course lesson và quiz không? | Có. Restrict được course, lesson, quiz, và custom LearnPress content types. |
| Đã có membership, upgrade có mất data không? | Không. Existing plans, members, course mappings giữ nguyên. |
| Có hỗ trợ Elementor không? | Restriction không break Elementor editor/preview. Elementor visibility conditions sẽ support trong version tương lai. |
| Khác gì so với WooCommerce Memberships? | LP Membership hiểu cấu trúc LearnPress native. WooCommerce Memberships không có LMS integration. |
| Có free trial không? | Có, qua WooCommerce Subscriptions. Hết trial auto-charge. |
